hexo博客中使用github作为图片的链接源

  • hexo

今天在用马克飞象写博客的时候，直接向博客中插入了截图，这个在本地运行很好，可以生成pdf和markdown文件。但是当将该博客提交到github上时，图片却没有显示。网上搜索了一些解决方法，pein0119的方法恰好解决了我的问题，现在总结一下，方便以后查阅。

  1. 首先在source目录下新建一个目录，如image，博客中需要插入的图片放到该目录下。执行hexo d时，hexo会将这些图片push到github上。

  2. 在博客中通过！[链接文字](/path/image)命令插入图片。假如我有一张图片在github上的地址为https://github.com/skycsliuyang/skycsliuyang.github.io/blob/master/img/1465786546512.png，在markdown文件中插入该图片就是用下面的命令：
    ![](https://github.com/skycsliuyang/skycsliuyang.github.io/blob/master/img/1465786546512.png?raw=true)

    注意：注意末尾的?raw=true.
